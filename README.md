#### Version 0.1.0 (unreleased)

Vimiv is an image viewer with vim-like keybindings. It is written in python3
using the Qt5 toolkit. Complete rewrite of the initial Gtk3 version.

## Dependencies
* python3
* python-pyqt5
* python-setuptools (for installation)
